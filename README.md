# Student Performance Prediction using Machine Learning

## Overview
This project is based on Artificial Intelligence and Machine Learning (AI/ML).
The main goal of this project is to predict student marks based on different factors like study hours, sleep, and attendance.

In this project, I tried to understand how daily habits affect academic performance and used a machine learning model to make predictions.

## Features
- Predict student marks based on input data
- Uses multiple factors (study hours, sleep, attendance)
- Data visualization using graphs
- Simple and easy-to-use command line interface
- Displays model accuracy (R² score)

## Machine Learning Concept Used
- Algorithm: Linear Regression
- Why Linear Regression?
  Because the output (marks) is a continuous value, and Linear Regression is suitable for predicting such values.
